## 九、核心业务流程测试 (E2E)

### 9.1 完整购买流程
```
登录 -> 浏览商品 -> 加入购物车 -> 结算 -> 支付 -> 查看订单
```

### 9.2 收藏-购买流程
```
浏览商品 -> 收藏 -> 从收藏列表进入 -> 加入购物车 -> 购买
```

### 9.3 订单取消流程
```
创建订单 -> 取消订单 -> 检查库存恢复
```
