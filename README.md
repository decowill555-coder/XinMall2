# XinMall（威星）

> 一站式数码生活服务平台 —— 从攻略探讨到购买的全链路解决方案

## 项目简介

XinMall 是一个整合 **酷安 + 闲鱼 + ZOL中关村在线** 核心功能的垂直数码服务平台，为用户提供从产品了解、社区讨论到交易购买的完整体验。

### 核心定位

| 平台 | 整合功能 | XinMall 对应模块 |
|------|----------|------------------|
| **酷安** | 数码社区、测评、攻略、讨论 | 论坛系统、帖子、话题 |
| **闲鱼** | C2C二手交易、个人开店 | 商品发布、店铺管理、订单交易 |
| **ZOL中关村在线** | 产品参数库、型号查询、价格对比 | 产品库、型号搜索、分类索引 |

### 目标用户

- 🎯 **数码爱好者**：寻找测评、分享经验、参与讨论
- 💰 **二手交易用户**：买卖二手数码产品
- 🔍 **选购咨询用户**：查询参数、对比价格、获取购买建议

## 核心功能

### 📱 内容社区

- **论坛系统**
  - 型号论坛：每个数码型号专属论坛，聚集同款用户
  - 用户论坛：用户自发创建的主题社区
- **帖子系统**
  - 测评分享、使用攻略、问题求助
  - 图文混排、标签分类
- **话题系统**
  - 热门话题、话题订阅、话题聚合

### 🛒 电商交易

- **商品管理**
  - 型号标准化：所有商品必须关联具体型号
  - 成色描述：全新/99新/95新/9新/8新/7新及以下
  - 多规格支持：容量、颜色、版本等
- **店铺系统**
  - 个人开店：C端用户轻松开店
  - 官方授权：品牌认证、正品保障
- **交易流程**
  - 购物车、订单管理、支付集成
  - 物流跟踪、评价系统

### 📊 产品库

- **型号搜索**
  - 智能搜索提示（型号 + 论坛）
  - 热门型号推荐
- **分类索引**
  - 设备类型分类（手机、电脑、平板等）
  - 字母索引快速定位
- **参数展示**
  - 详细规格参数
  - 型号对比功能


## 开发指南

### 环境要求

- Node.js >= 16.0.0
- Java 21+
- MySQL 8.0+
- Maven 3.6+

### 前端启动

```bash
cd FE

# 安装依赖
npm install

# H5 开发
npm run dev:h5

# 微信小程序开发
npm run dev:mp-weixin

# App 开发
npm run dev:app

# 构建
npm run build:h5
npm run build:mp-weixin
npm run build:app
```

### 后端启动

```bash
cd BE/XinMall

# 编译
mvn clean install

# 运行
mvn spring-boot:run
```

## 设计系统

### 主题支持

- **浅色模式**：活力橙色系（#FF6A00）
- **深色模式**：生物荧光感设计（霓虹紫/赛博朋克风格）

### 组件规范

采用 **原子设计（Atomic Design）** 理念：

- **Atoms（原子）**：UiButton、UiIcon、UiTag、UiPrice 等
- **Molecules（分子）**：UiSearch、UiInput、UiNavbar 等
- **Organisms（有机体）**：UiGoodsCard、UiFilterSidebar 等
